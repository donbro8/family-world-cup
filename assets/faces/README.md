# Family faces

Drop a square-ish photo for each person here, named after their key in `data.json`:

```
mom.png  dad.png  lee.png  maddie.png  becca.png
donnie.png  melissa.png  geoff.png  mercia.png
```

- Any size works — the site crops to a circle automatically (`object-fit: cover`).
- `.png`, `.jpg`, or `.webp` are all fine (match the extension in `data.json`'s `face` field).
- **Missing a photo?** No problem — the site shows a coloured circle with the person's initial as a fallback, so it always looks intentional.
